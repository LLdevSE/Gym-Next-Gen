# Gym Next Gen

Gym Next Gen is a state-of-the-art, AI-driven fitness facility management and e-commerce platform. It features a polyglot microservices architecture with a Node.js/Express backend and a Python ML service for personalized fitness and dietary recommendations.

## Tech Stack

- **Frontend:** React.js (Vite), TailwindCSS, Framer Motion
- **Backend:** Node.js, Express.js, MongoDB (Mongoose)
- **ML Service:** Python (FastAPI/Flask), Scikit-Learn (Random Forest, KNN)
- **Authentication:** JWT (JSON Web Tokens)
- **File Storage:** Cloudinary

## Features

- **Admin Dashboard:** Manage coaches, customers, and gym store inventory.
- **Coach Module:** Manage profile, availability, and session bookings.
- **Customer Module:** E-commerce gym store, coach booking system, and AI-driven fitness recommendations.
- **NextGen AI Oracle:** Personalized workout and nutritional plans based on user physical data.

## Project Structure

- `/frontend`: React application (Vite).
- `/backend`: Node.js Express server.
- `/ml-service`: Python ML microservice.
